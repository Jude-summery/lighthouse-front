### 1. 登录模块
#### 1.1 账号登录
**接口地址**
```
/login/signin
```
**必选参数**
```
account: string //账号
password: string //密码
```
**可选参数**
```
rememberLogin: boolean //30天免登陆
```
**调用示例**
```
POST /login/signin
@body: 
{
    account: "*******",
    password: "******",
    rememberLogin?: true
}
```
#### 1.2 刷新登录
**接口地址**
```
/login/refresh
```
**调用示例**
```
GET /login/refresh

@response:
{
    body: {
      nickname: string,
      userId: number,
      status: number // ['离线', '在线']
    }, // 无返回内容则返回 null
    code: number, // 状态编码
    success: boolean, // 状态
    message: string, // 错误消息
}
```
#### 1.3 获取登录状态
**接口地址**
```
/login/status
```
**调用示例**
```
GET /login/status
@response:
{
    body: {
      status: number // ['离线', '在线']
    }, // 无返回内容则返回 null
    code: number, // 状态编码
    success: boolean, // 状态
    message: string, // 错误消息
}
```
#### 1.4 退出登录
**接口地址**
```
/logout
```
**调用示例**
```
GET /logout
```
### 2. 用户模块
#### 2.1 获取用户信息
说明：登录后调用此接口，可以获取用户详情
**接口地址**
```
/user/info
```
**调用示例**
```
GET /user/info

@response:
{
    body: {
      gender: number, // 性别 ['保密', '男性', '女性']
      birthday: number, // 出生日期 时间戳
      nickname: string, // 昵称
      provice: number, //省份id
      city: number, //城市id
      signature: string, //用户签名
      groupId: number, //用户组id，多个用
      groupRole: number, //用户角色
      status: number, // ['离线', '在线']
    }, // 无返回内容则返回 null
    code: number, // 状态编码
    success: boolean, // 状态
    message: string, // 错误消息
}
```
#### 2.2 新增用户（注册）
**接口地址**
```
/user/signup
```
**必选参数**
```
nickname: string //昵称
account: string //账号
password: string //密码
```
**调用示例**
```
POST /user/signup
@body: 
{
    nickname: "*******",
    account: "******",
    password: "******"
}
```
#### 2.3 获取用户动态
**接口地址**
```
/user/event
```
**可选参数**
```
pageSize: number //单页长度
pageNo: number //页码
```
**调用示例**
```
GET /user/event

@response:
{
    body: {
        results: [
            {
                eventId: number, //动态Id
                eventContent: string, //动态详情
                eventTime: number, //动态时间
            }
        ]
    }, // 无返回内容则返回 null
    code: number, // 状态编码
    success: boolean, // 状态
    message: string, // 错误消息
}
```
#### 2.4 获取用户任务列表
**接口地址**
```
/user/tasks
```
**可选参数**
```
pageSize: number //单页长度，默认10
pageNo: number //页码
```
**调用示例**
```
GET /user/event

@response:
{
    body: {
        results: [
            {
                taskId: number, //任务Id
                taskName: string, //任务名称
                taskContent: string, //任务内容
                taskType: number, //任务分类
                taskPlanEndTime: number, //计划结束时间
                taskEndTime: number, //任务结束时间
                creatorId: string, //创建人id
                creatorName: string, //创建人昵称
                createTime: number, //创建时间
                executorId: number, //执行人id
                executorName: number, //执行人昵称
                executeTime: number, //接手任务时间
                process: string, //任务流转轨迹，用户id，使用逗号分隔
            }
        ]
    }, // 无返回内容则返回 null
    code: number, // 状态编码
    success: boolean, // 状态
    message: string, // 错误消息
}
```
