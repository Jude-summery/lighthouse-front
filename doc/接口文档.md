### 1. 登录模块
#### 1.1 账号登录
**接口地址**
```
/login/signin
```
**必选参数**
```
account: string //账号
password: string //密码
```
**可选参数**
```
rememberLogin: boolean //30天免登陆
```
**调用示例**
```
POST /login/signin
@body: 
{
    account: "*******",
    password: "******",
    rememberLogin?: true
}

@response:
{
    body: null,
    code: ,
    success:,
    message:,
}
```
